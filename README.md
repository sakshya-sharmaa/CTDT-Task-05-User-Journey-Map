# CTDT-Task-05-User-Journey-Map
Task 05 – User Journey Map

Product

Samsung AI Family Hub Smart Fridge

User Journey Stages

Awareness

- User sees advertisements and social media content about the AI Smart Fridge.
- Becomes interested in smart kitchen technology.

Research

- Reads online reviews and product specifications.
- Compares features with other smart refrigerators.

Purchase

- Purchases the AI Family Hub Smart Fridge.
- Expects convenience and intelligent features.

Setup

- Connects the fridge to Wi-Fi.
- Configures accounts and preferences.

Daily Usage

- Stores food items and monitors inventory.
- Uses touchscreen and smart features regularly.

Smart Features Experience

- Receives recipe recommendations.
- Uses AI-powered food management and smart home integration.

Long-Term Satisfaction

- Enjoys convenience and efficiency.
- Benefits from software updates and improved functionality.
